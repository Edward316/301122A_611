Objetivo general:


Se espera que las personas que ingresen a este OVI comprendan la importancia, aplicación y prevención de riesgos informáticos.


Objetivo específico:


1. Conocer conceptos básicos de seguridad informática.

2. Agrupar conocimiento para evitar riesgos de seguridad.

3. Definir impactos si no se protege de manera adecuada.
